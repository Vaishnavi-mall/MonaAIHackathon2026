# Mona AI Hackathon

Professional monorepo scaffold for ten Gemini-powered AI agent prototypes.

## Setup

1. Create and activate a Python 3.12 virtual environment.
2. Run `make install`.
3. Copy `.env.example` to `.env` and add your Gemini API key.
4. Run `make run`.

Open <http://localhost:8000> to view the demo interface. Agent modules are placeholders and contain no implementation yet.
